# TicTacToe
Simple <b>Tic Tac Toe</b> game written using <b>HTML</b>, <b>CSS</b>, <b>Javascript</b> and <b>jQuery</b>.<br>
There are two variants of the game:
<ul>
<li>Player vs Player</li>
<li>Player vs Computer</li>
</ul>
